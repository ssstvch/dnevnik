# Запуск проекта

    Для запуская проекта потребуется установленный Node.js и npm.
    Находясь в папке проекта "source" в IDE, в терминале ввести команду
        npm install
    чтобы загрузить все зависимости. После успешной загрузки, не выходя из папки в терминале ввести команду    
        npm start 
    При необходимости, согласиться сменить порт.
